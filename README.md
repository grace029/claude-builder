A campus discovery app for Princeton students to explore hidden gems and collect tokens.

## Features

- **Discover Locations**: Explore 8 unique, undiscovered places across Princeton's campus
- **Collect Tokens**: Earn 10 tokens for every location you visit
- **Track Progress**: Monitor your exploration journey with achievements and milestones
- **Level System**: Progress from "New Explorer" to "Master Explorer"

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
```

## How It Works

1. Browse through curated campus locations
2. Visit a location in person
3. Mark it as visited to collect tokens
4. Unlock achievements and level up!

## Technology Stack

- React 18
- CSS3 with animations
- Local state management

## Campus Locations

TigerTrails features 8 unique locations across campus:
- Secret Garden at Prospect House
- Firestone Library Roof Garden
- Campus Tunnel System
- Behind the Graffiti Wall
- The Sundial at McCosh Walk
- Stony Ford Cedar Swamp
- East Pyne Courtyard
- The Kissing Rock
